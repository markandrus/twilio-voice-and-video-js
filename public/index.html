Tokens

Choose an identity and click "Create" to create a Capability and Access Token pair. twilio.js will use the Capability Token to make and receive voice calls, while twilio-video.js will use the Access Token to connect to Rooms.

Voice

Choose an identity and click "Call" to place an outgoing call to the specified Client using twilio.js. Incoming calls will be accepted automatically.

Video

Choose a Room name and click "Connect" to connect to the specified Room using twilio-video.js.
